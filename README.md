## Descrição do projeto 

<p align="justify">
  O projeto é um pequena aplicação de jogo da velha para se jogar no terminal.
</p>

## Funcionalidades

:heavy_check_mark: Nomes inseridos passam pela validação, impedindo que nomes iguais sejam inputados.

:heavy_check_mark: Posições inseridas via input são validadas para apenas valores numéricos serem preenchidos.

:heavy_check_mark: Cada jogada é feita inserindo uma posição na fileira e no quadrante, em valores de 1 a 3.

## Pré-requisitos

:warning: [Node](https://nodejs.org/en/download/)

:warning: [prompt-sync](https://www.npmjs.com/package/prompt-sync)


## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto:
```
git clone git@github.com:mininoterraria/Jogo-da-Velha.git
```
Instale as bibliotecas necessárias:
```
npm install prompt-sync
```
Utilize o seguinte comando para iniciar o jogo:
```
node src\index.js
```
